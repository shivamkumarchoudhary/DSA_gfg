<h2><a href="https://www.geeksforgeeks.org/problems/remove-duplicate-element-from-sorted-linked-list/1">Remove Duplicates from a Sorted Linked List</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 14pt;">Given the head of a <strong>sorted</strong> singly linked list, remove all duplicate nodes so that each element appears only once. The resulting linked list should remain sorted.</span></p>
<p><span style="font-size: 14pt;"><strong>Note:</strong> Try to solve the problem without using extra space.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 12pt;"><strong>Input: </strong>Head: 2-&gt;2-&gt;4-&gt;5
<strong>Output: </strong>2 -&gt; 4 -&gt; 5<br><strong><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926429/Web/Other/blobid1_1778504637.webp" width="387" height="129"><br>Explanation: </strong>In the given linked list 2 -&gt; 2 -&gt; 4 -&gt; 5, only 2 occurs more than 1 time. So we need to remove it once.
</span></pre>
<pre><span style="font-size: 12pt;"><strong>Input: </strong>Head: 2-&gt;2-&gt;2-&gt;2-&gt;2
<strong>Output: </strong>2<br><strong><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926429/Web/Other/blobid2_1778504663.webp" width="387" height="129"><br>Explanation: </strong>In the given linked list  2 -&gt; 2 -&gt; 2 -&gt; 2, 2 is the only element and is repeated 5 times. So we need to remove any four 2.</span></pre>
